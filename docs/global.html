Methods

detectApiVersionMiddleware()

Detects the API version that's been requested, either from the N-Api-Version header or the URL parameters.

errorHandler()

Configures the app with an error handler. A message and userMessage field is added to the error's json body. The userMessage is a translated version of the message field if translation is properly configured for this middleware. You can set both fields explicitly in the error response by setting err.data to the desired object.

hc()

Exposes a public endpoint for health checks.

httpError()

Convenient wrapper to throw an error that has an HTTP status code. These errors are public-friendly, meaning their message can be displayed on the API.

start()

Starts an http express server except on testing enviroment

staticPath()

Exposes a static files based on passed path
